Admin mode reference

Customizing page contents

Admin mode allows you to add self-made content on the top and bottom page areas. Use this feature to provide your own banners, web-counters, site logos or custom site enhancements.

Antarcticle uses codepen.io as an external, feature rich custom content editor.

To customize your Antarticle instance follow these simple steps:

1. Go to signup page and choose a free plan

2. Create a new Pen

3. Compose any content you like. With embedded preview you can see rendered result as you type

4. Now press [Save]. Your pen is being assigned a unique URL. Use this URL in Antarcticle

5. Use URL from step [4] to embed your content into Antarcticle. From now on every change made to this Pen on codepen.io will be applied to Antarcticle automatically
